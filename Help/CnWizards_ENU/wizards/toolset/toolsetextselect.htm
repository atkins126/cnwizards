Extending Select Tool

This tool is designed to progressively expand the selection range layer by layer within the currently selected block of source code. It supports Delphi/C++Builder and allows for customizable hotkeys.

You can place the cursor at the desired location for selection before use. By continuously pressing the hotkey, users can achieve layer-by-layer expansion of the selected area.
